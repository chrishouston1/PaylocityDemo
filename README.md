# Paylocity Demo

#### Live demo: https://paylocitydemo.herokuapp.com
****
This project is a coding challenge for Paylocity. Go to the demo website and either sign into an existing company account or create your own. After signing in, you can view existing employees or create new ones. To get a preview of costs of the benefits package, go to the Calculate page and select an employee.
