# Paylocity Demo

#### Live demo: https://paylocitydemo.herokuapp.com
****
This project is a coding challenge for Paylocity. Go to the demo website and either sign into an existing company account or create your own. After signing in, you can view existing employees or create new ones. Click on the calculate button next to the employee to get a preview of the benefits costs.
